# PlantaFix

![Planta jovem](https://terramagna.com.br/wp-content/uploads/2022/09/Planta-jovem-luz-solar.jpg)

**PlantaFix** é um projeto de feira de ciências com o objetivo de auxiliar na identificação de plantas medicinais com base em doenças ou sintomas informados pelo usuário.  
O sistema também oferece informações sobre cultivo, localização geográfica e formas de uso de cada planta sugerida.
