# PlantaFix
![foto]([URL-da-imagem](https://terramagna.com.br/wp-content/uploads/2022/09/Planta-jovem-luz-solar.jpg))

Projeto de feira de ciências onde o usuário fala o nome de uma doença ou sintoma, e o programa responde qual planta pode curar. Além disso, mostra como cultivar a planta, onde achar, etc...

pra rodar apenas:
```bash
git clone https://github.com/Str44ng33/PlantaFix/
cd PlantaFix
npm install
npm start
```
## Como contribuir para o PlantaFix

1. Faça um fork do projeto.
2. Clone seu fork: `git clone https://github.com/Str44ng33/PlantaFix.git`
3. Crie uma branch para sua alteração: `git checkout -b minhabranch`
4. Faça suas modificações, salva e depois:  
   `git add .`  
   `git commit -m "qualquer coisa"`
5. Envie sua branch para o github: `git push origin minhabranch`
6. Abra um Pull Request aqui no repositório original.

Antes de enviar o PR, mantenha seu fork atualizado:  
`git remote add upstream https://github.com/Str44ng33/PlantaFix.git`  
`git pull upstream main`

Pronto, assim é simples contribuir
