# PlantaFix
![Planta jovem](https://terramagna.com.br/wp-content/uploads/2022/09/Planta-jovem-luz-solar.jpg)

<p style="color:green;">
Projeto de feira de ciências onde o usuário fala o nome de uma doença ou sintoma, e o programa responde qual planta pode curar. Além disso, mostra como cultivar a planta, onde achar, etc...
</p>

<p style="color:green;">
Pra rodar apenas:
</p>

```bash
git clone https://github.com/Str44ng33/PlantaFix/
cd PlantaFix
npm install
npm start
```
